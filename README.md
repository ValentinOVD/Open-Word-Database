# How to test ?
  You can test Open Word Database by using console.sh :
./console.sh

po

yes

translate

1

fr

hello

		
You can also use gettext for translating word.  
		
For that, install the mo files :

./console.sh

mo

yes

yes

And after you can use gettext : echo "\`gettext fr computer\`"
